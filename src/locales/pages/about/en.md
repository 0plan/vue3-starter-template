**VueForge** is a refined evolution of modern frontend tooling. It combines the power of **Vite**, the flexibility of **Vue 3**, and the aesthetic excellence of **shadcn-vue** to help you build stunning applications faster than ever.

### Key Features

- ⚡️ **Vite 7** - Next generation frontend tooling
- 🟢 **Vue 3** - Composition API & Script Setup
- 🎨 **Tailwind CSS v4** - The utility-first CSS framework
- 🧩 **shadcn-vue** - Beautifully designed components
- 📂 **File-based Routing** - Organized and intuitive
- 📦 **Components Auto Importing** - Efficiency at its core
- 🌍 **I18n & PWA Support** - Ready for the world and the web

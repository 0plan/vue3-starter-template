**VueForge**는 현대적인 프론트엔드 도구의 정교한 진화입니다. **Vite**의 강력함, **Vue 3**의 유연함, 그리고 **shadcn-vue**의 심미적 우수성을 결합하여 그 어느 때보다 빠르게 멋진 애플리케이션을 구축할 수 있도록 돕습니다.

### 주요 특징

- ⚡️ **Vite 7** - 차세대 프론트엔드 도구
- 🟢 **Vue 3** - Composition API 및 Script Setup 지원
- 🎨 **Tailwind CSS v4** - 유틸리티 우선 CSS 프레임워크
- 🧩 **shadcn-vue** - 아름답게 디자인된 컴포넌트 프리셋
- 📂 **파일 기반 라우팅** - 직관적인 프로젝트 구조
- 📦 **컴포넌트 자동 임포트** - 개발 효율성 극대화
- 🌍 **I18n 및 PWA 지원** - 글로벌 및 웹 표준 준비 완료

```ts
// VueForge - 현대적인 개발 경험
const vueforge = {
  speed: '번개처럼 빠름',
  beauty: '단단한 완성도',
  dx: '우수한 개발자 경험'
}

console.log('VueForge와 함께 더 날카롭게 만들어가세요!')
```
